# Vishal Yadav — Portfolio

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Visit%20Portfolio-7c3aed?style=for-the-badge&logo=render)](https://vishal-portfolio-mezw.onrender.com)

**Live:** [https://vishal-portfolio-mezw.onrender.com](https://vishal-portfolio-mezw.onrender.com)

A modern, responsive portfolio website built with **Django** + **Tailwind CSS**.

## Features

- Animated hero with typing effect & floating tech badges
- Glassmorphism UI with dark theme
- Skills with animated progress bars (tabbed)
- Projects showcase with hover overlays
- Experience & Education timeline
- Achievements & Certifications
- Working contact form (saves to DB)
- Custom cursor, scroll reveal animations
- Fully responsive (mobile + desktop)

## Tech Stack

- **Backend:** Python, Django 5.2
- **Frontend:** Tailwind CSS (CDN), Vanilla JS
- **Database:** SQLite
- **Icons:** Font Awesome 6

## Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/vishal-portfolio.git
cd vishal-portfolio

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run migrations
python manage.py migrate

# 4. Create superuser
python manage.py createsuperuser

# 5. Run server
python manage.py runserver
```

Open **http://127.0.0.1:8000**

Admin panel: **http://127.0.0.1:8000/admin**

## Add Your Data

Go to `/admin` and add:
- Profile (name, bio, photo, links)
- Skills with percentages
- Projects
- Experience & Education
